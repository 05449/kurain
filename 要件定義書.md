# クイズアプリケーション要件定義書

## 1. 概要
指定されたクイズデータ（Excelファイル）を使用して、Webブラウザで動作するクイズアプリケーションを開発する。

## 2. 機能要件

### 2.1. 画面構成
本アプリケーションは以下の3つの画面で構成される。
- タイトル画面
- クイズ画面
- リザルト画面

### 2.2. タイトル画面
- アプリケーションの初期画面として表示される。
- 「START」ボタンを配置する。
- 「START」ボタンをクリックすると、クイズ画面に遷移する。

### 2.3. クイズ画面
- クイズデータから問題と選択肢をランダムに表示する。
- ユーザーが選択肢を選択すると、正解・不正解の判定を行う。
- 判定後、正解と解説を表示する。
- 全ての問題に回答すると、リザルト画面に遷移する。

### 2.4. リザルト画面
- クイズの結果（正解数など）を表示する。
- ランキングを10位まで表示する。
- 「RETRY」ボタンを配置する。
- 「RETRY」ボタンをクリックすると、タイトル画面に戻る。

### 2.5. ランキング機能
- クイズのスコアを記録し、上位10件を表示する。
- スコアはブラウザのローカルストレージに保存し、永続化する。

## 3. 非機能要件

### 3.1. 動作環境
- モダンなWebブラウザ（Google Chrome, Firefox, Safari, Edgeなど）で動作すること。

### 3.2. 使用技術
- HTML
- CSS
- JavaScript

## 4. データ
- クイズの問題、選択肢、正解、解説は提供された`quiz.xlsx`ファイルの内容を使用する。
